Python Variable Types

Variable Type

What it stores

int

Stores an integer (a number with no decimal places)

str

Also known as a string, stores text

bool

Also known as boolean, stores values of True or False

float

Stores a deciaml to 8 decimal places

list

Also called an array see here

decimal

Stores a deciaml to a defined amount of places decimal places, see the python wiki for details on this
