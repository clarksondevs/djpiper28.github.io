Python Errors

Common Causes:

Incorrect syntax, wrong variable types being used as parameters

Error Name

How To Fix

Memory Error

List is too long or int is not a 64 int

TypeError: unsupported operand type(s) for +: 'int' and 'str'

Turn your string to an int with int(yourStringHere)

TypeError: x() takes 0 positional arguments but x were given

Remove some arguments from the declaration of the method
