Python if, elif and else statements

Brackets:

Brackets for the if and elif statements are required, but with the else statement it is different e.g:

if("a"==6):

    print("a is the same as 6")

elif("a"=="a"):

    print("a is the same as a")

else:

    print("a is not 6 or a")
