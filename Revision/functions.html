Python Methods/Functions

Create a method/function:

Use def [method name]([method parameters e.g: number, word]):[what to do with the data] e.g:

def Hello(name):

    print("Hello" + name)

Return Data:

This means once the method is called it sends a message back to you. Use return [data]

Call a Method:

Type [methodname]([parameters]) e.g:

Hello(input("What is your name?"))
