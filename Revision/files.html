Python Files

Opening The File:

All files must be opened and assigned to a variable in python e.g:

f = open("users.txt", [file access method (see below) such as "w"])

File Access Methods

All files must have an opening method, see this table for details

Opening method

What it does

w

Write mode for the files - rewrites all data when the write method is called

w+

Creates a new file

a

Append mode - adds data to the end of a file, does not add a new line

r

Reads the file, see below for usage

Writing to The File:

Use the method write(string) it only accepts stings, if you want the data to be added to a new line then add "\n" before your text how the text is added depends on the opening method above, also in read mode this method won't work. e.g:

f = open("users.txt", "a")

f.write("\nText")

Reading from The File:

f = open("users.txt", "r")

for line in f:

    print(line)

Closing The File:

f.close

Notes

The variable f is the variable that the file is opened with so you can use any variable you want to use.
