Python Arrays/Lists

Declaring an array:

This is done so that you don't create a string or cause errors later on e.g:

variableName = []

Adding data to an array:

This is done with append() this accepts arrays, ints, bools, floats and any variable types; each thing added is an element (to the array) e.g:

variableName.append("Element One")

Reading data from an array:

You can use:

variableName[index of element]

An index is the element number - 1 as the numbers are stored in binary and start at 0000000000000000... (64 zeros) or you can use:

for element in variableName:

    print(element)

Removing data from an array:

variableName.remove([element value])
