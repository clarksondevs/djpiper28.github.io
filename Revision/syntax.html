Python Syntax

Brackets:

Brackets in python must in the correct place because otherwise it will cause errors. When using a method always use brackets after the method name e.g:

print("Hello World")

Commenting:

In Python you can comment code by using a hash tag (#) this means that the text after the hash tag will not be ran like code e.g:

print("Hello World"*5) #Prints Hello World five times

Indentation:

In python after a colon the text below (nested) should be indented do this by pressing space four times or pressing Tab.

Numbers:

When you are using numbers and one of the numbers is a string make sure you turn it to an int before hand or there will be an error or the string will repeat itself many times e.g:

number = input("Insert a number. ")

print(1 + int(number)

Parameters:

All Python methods take parameters, these define what it should do e.g:

f = open("file.txt", "r") #The method open takes two parameters (at least) one for the filename and one for the open type
