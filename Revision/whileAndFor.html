Python While and For Loops

While:

This runs the code nested in it whilst the condition is true e.g:

while(1==1):

    print("One is always equal to one so this is an infinite loop")

For:

This runs the code nested in it whilst the condition is true BUT there must be a variable e.g:

for(2>one):

    print("Two is always greater than one (The variable is one) so this is an infinite loop")

OR! it goes through each element of an array e.g:

for element in ["array", "element"]:

    print(element)

OR! it goes through a variable for a value e.g:

for "a" in "aaaaaaaaaa":

    print("found an a")
